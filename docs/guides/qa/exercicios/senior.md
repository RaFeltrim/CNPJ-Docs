# Exercícios — Sênior

Objetivo: liderar estratégia de qualidade, métricas e conformidade.

Tarefas:

1. Criar um plano de adoção Shift Left para um produto (roadmap 3 meses) com milestones, responsáveis e KPIs (MTTR, cobertura, flakiness).
2. Mapear riscos de LGPD e desenhar políticas de logging/mascaramento para testes (base: `plano-estudo.md` dia 6).
3. Projetar arquitetura de observabilidade (logs, tracers, métricas) para testes automatizados.
4. Avaliar ROI de automação e propor priorização de features para automação.
5. Entrega: documento estratégico + apresentação de 15 minutos.

Critério de aceitação:
- Roadmap com KPIs e plano de execução aprovado pelos stakeholders.

Tempo estimado: 1–2 semanas.
